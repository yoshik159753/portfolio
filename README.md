# Portfolio

ポートフォリオサイトです。

<https://portfolio.pleasurecode.jp/>

## 技術スタック

- Next.js 15
- Tailwind CSS 4.x

## 開発コマンド

```bash
# 依存パッケージのインストール
npm install

# 開発サーバー起動
npm run dev
```
